# NexusOs
NexusOS — это полнофункциональная веб-операционная система, разработанная на чистом HTML, CSS и JavaScript. Она имитирует работу классической ОС Windows с современным дизайном, включая систему аккаунтов, рабочий стол с приложениями, перетаскиваемые окна и полностью адаптивный интерфейс для мобильных устройств.
